# 地域未来交付金（地域未来推進型）申請書作成ノウハウ

## 1. 制度の目的
地方の大きな伸び代と地域特性を最大限に活かし、地方の暮らしの安定を実現するとともに「強い経済」を構築するため、**地場産業の付加価値向上など、地方公共団体の地域独自の取組を幅広く支援**する交付金である。従来の地方創生に資する取組のみならず、各自治体による**産業クラスター計画や地場産業の成長戦略**が、真に地方の活力を最大化することに繋がるような取組を推進する。

※デジタル実装型（TYPE A/V/S）が「デジタル実装による課題解決」に特化するのに対し、地域未来推進型はデジタルに限らず**地域独自の産業・雇用・関係人口・暮らしの取組を幅広く支援**する点が特徴。

## 2. 事業区分・補助率・国費上限
| 事業区分 | 事業計画期間 | 交付上限額（国費・1自治体当たり） | 補助率 |
|---|---|---|---|
| ソフト事業 | 原則3か年度以内（最長5か年度） | 都道府県15億円／中枢中核15億円／市区町村10億円（/年度） | 1/2 |
| 拠点整備事業 | 原則3か年度以内（最長5か年度） | 都道府県15億円／中枢中核15億円／市区町村10億円（/年度） | 1/2 |
| インフラ整備事業 | 原則5か年度以内（最長7か年度） | 事業計画期間中の総国費：都道府県50億円／中枢中核20億円／市区町村10億円（単年度目安10/4/2億円） | 1/2等（各省庁の交付要綱に従う） |

**主なルール**
- 単年度の交付上限額は目安。
- 新規事業の申請上限件数は自治体規模を問わず原則10件。
- インフラ整備事業は、ソフト事業又は拠点整備事業との組み合わせを要件とする。
- ソフト＋ハードや分野間連携の事業も、一本の申請で受付（申請の効率化）。
- 広域リージョンとして複数自治体で実施する事業は、1リージョンあたり申請可能な事業数は最大5事業・交付上限額（国費）最大10億円/年度。自治体が広域リージョンとして申請できる事業数は最大3事業まで。

## 3. 評価基準（S〜Dの5段階評価）
以下の4観点で評価される。申請書は各観点で高評価を得られるよう記述する。
1. **目指す将来像及び課題の設定** … 地方版総合戦略に基づき、目指す将来像と構造的課題を、人口・産業データ等の客観的根拠とともに明確化。
2. **KPI設定の適切性** … アウトカム（住民・地域への効果）を示す必須KPIを設定し、複数年度で費用対効果を計測できること。
3. **自立性** … 事業推進主体が将来的に本交付金に頼らず継続できる仕組み（自主財源の確保）を示すこと。
4. **地域の多様な主体の参画** … 産・官・学・金・労・言・住民全般が事業の検討・実施・検証の各段階に参画し、効果検証・改善・公表を行うこと。

## 4. 実施計画の出力フォーマット（全13セクション）
申請書（実施計画）は以下の構成で作成する。各項目名はこの表現を用いること。

1. **申請者情報** … 都道府県／市区町村／地方公共団体コード／事業開始年度／継続・変更／担当部局・責任者・担当者・連絡先。
2. **交付対象事業の名称等** … 交付対象事業の名称／単独・広域の別／主となる事業分野（大項目・詳細）／関連する事業分野①②／ソフト事業・拠点整備事業・インフラ整備事業の各実施期間／各事業の対象事業経費・申請額（対象経費の1/2）。
3. **交付対象事業の背景・概要**
   - **Ａ．地方創生として目指す将来像（交付対象事業の背景）** … 【事業の背景】と【目指す将来像】を記述。
   - **Ｂ．地方創生の実現における構造的な課題** … 人口動態・労働力率・創業比率・産業構造・関係人口等の客観的データで課題を明確化。
   - **Ｃ．交付対象事業の概要** … 交付対象事業がどのように構造的課題の解決に寄与するのかが分かるように記載。個別事業（例：①スキリング事業 ②創業スタートアップ支援 ③女性活躍応援 ④シティプロモーション等）を列挙。
4. **重点テーマ／優先採択テーマへの該当** … 該当有無と重点テーマ名（例：若者・女性にも選ばれる地方）。
5. **重点配分事業への該当** … 他省庁補助金との戦略的連携／他施策との連携による弾力措置／特区制度の活用／地域再生法に基づく事業／スタートアップ支援に係る事業。
6. **事業の横展開の状況** … 横展開の該当有無／参考にした自治体および政策分野（最大5団体）。
7. **他の国庫補助金等の対象事業の除外** … ①財源振替・経常経費・本来一般財源で措置すべき経費の除外 ②提案・企画立案のコンサルティング経費の除外 ③他の国庫補助金・特別交付税等の対象部分の除外、をチェック。
8. **交付対象事業の重要業績評価指標（KPI）** … KPI①〜⑥。**KPI①は事業・施策の全体効果を示す必須KPI（アウトカムベース）**とする。各KPIに単位・各年度増加分・累計・目標値の根拠・計測手法を記載。進捗や効果検証を踏まえた事業の見直し内容も記載。
9. **自立性** … 事業推進主体が将来的に本交付金に頼らず継続できる取組内容／自主財源の種類【A】一般財源【B】寄附金（ふるさと納税・企業版ふるさと納税等）【C】〜【E】と内容／各年度の自主財源見込額／交付対象事業経費（うちソフト事業費・うち拠点整備事業費）・総事業費。
10. **地域の多様な主体の参画** … 産・官・学・金・労・言・住民全般の各主体名／主な役割／意見及び改善方策への反映。
11. **交付対象事業の効果検証及び事業内容の見直しの方法・時期・体制** … 外部組織による検証（検証時期・検証方法・外部組織の参画者・検証結果の公表方法）。
12. **地方版総合戦略の策定状況** … 計画期間／基本目標。
13. **地域再生計画の名称等** … 名称／区域／計画期間。

（別紙2-1）**ソフト事業経費内訳** … 年度別×要素事業別に、事業名・概要・関連KPI・委託料等の内訳を記載。

## 5. 採択される申請書の書き方のポイント（記述の勘所）
- **Ｂ．構造的な課題**は必ず客観データで語る。例：国勢調査の若年女性労働力率、地域経済分析システム（RESAS）による創業比率の全国・県平均との比較、商工会員数の推移、若年女性人口の減少、将来人口推計など。「〜が急務となっている」と課題を明確化する。
- **Ｃ．事業概要**は複数の個別事業を束ねて相乗効果を出す。各事業を「①②③…」で列挙し、それぞれ対象者・手法・目指す状態を1〜2文で示す。
- **KPI**はアウトカム（創業件数、就業率、参加者の意識変化の割合、関係人口等）を必須KPIに据え、目標値の根拠（初年度◯人・翌年度以降◯%増 等）と計測手法を必ず添える。
- **自立性**は「4年目以降は交付金に頼らず継続」を明記し、一般財源（行財政改革による捻出）・ふるさと納税・企業版ふるさと納税・寄附金などの自主財源を具体的に示す。
- **多様な主体の参画**は産官学金労言住民の分類ごとに主体名（商工会・中小企業診断士協会・近隣大学・金融機関・労働組合・地元新聞社・住民/女性等）と役割・改善方策への反映を書く。
- 文体は「である」調。数値・固有名詞を具体的に。
