# 地域防災緊急整備型（災害時備蓄資材等整備事業）実施計画書 作成ノウハウ

## 1. 制度の目的
能登半島地震等の教訓を踏まえ、**避難生活環境の抜本的な改善**と**災害関連死の防止**を目的に、地方公共団体が行う**防災・減災資機材や災害対応車両の緊急的な整備**を支援する交付金である。単なる備蓄の積み増しではなく、「生命維持中心の備蓄」から「健康と尊厳が守られる避難所環境」への転換を図り、平時の利活用と住民の防災意識向上まで一体で進める取組を推進する。

## 2. 必須要件（1つでも欠けると審査対象外）
次の4項目は加点項目ではなく**必須要件**であり、1つでも満たさないと審査対象にならない。
1. 地域課題を踏まえ、従来の防災・減災対策を**改善**する事業であること。
2. 避難生活環境など、**地域住民の安心・安全に直接裨益**すること。
3. **継続的に事業を推進できる庁内・地域の連携体制**があること。
4. 資機材の**平時利用や周知広報**により、住民の防災意識を高めること。

さらに次の条件を満たすこと。
- 何らかの**資機材（車両含む）の購入**を伴うこと（外部委託費だけの事業は不可）。
- **車両・資機材の購入費が総事業費の50％を超える**こと。
- 行政事務だけに利用する事業・調査/実証だけの事業は不可。

## 3. 対象経費・対象外経費
**対象になり得る経費**
- 防災・減災資機材の購入（パーティション、簡易ベッド、可搬型空調機器、簡易トイレ、発電機、トイレカー、キッチンカー等）
- 災害対応車両の購入
- 今回購入する資機材を収納する倉庫の整備・改築（保管に必要なものに限る）
- 民間事業者・NPO等への資機材購入補助（間接補助。発災時協力を協定で担保）
- 外部アドバイザーの招聘／新規の防災体験イベント／操作・設営訓練／要配慮者参加型避難訓練
- 資機材を活用する避難計画・地区防災計画の作成／住民向け広報・プロモーション

**対象外経費（内閣府資料に明記）**
- 水・食料・燃料、取得価格10万円未満の備品・消耗品、使い捨て携帯トイレ、カセットボンベ、毛布・寝袋、マスク・消毒液・土のう
- パソコン・スマートフォン・タブレット・テレビ等、自治体職員の人件費・旅費
- 従来から実施している防災イベント、基金積立金、避難所建物そのものの改修・施設整備、用地取得・造成・土地借上げ
- 国の他補助金と重複する経費、基準に適合しない災害対応車両、快適トイレ仕様に適合しない仮設トイレ、調査・実証のみの事業
- ※「取得価格10万円未満」は備品単位で判定される可能性があり、都道府県への事前相談が必要。

## 4. 審査評価項目（S・A・B・C・Dの5段階）
| 評価項目 | 主な審査内容 |
|---|---|
| 地域課題 | 現状・不足・災害リスクが具体的か（数値化されているか） |
| 目指す防災力 | 整備による改善内容（整備前後の比較・展開時間・提供人数）が明確か |
| 平時利用 | 利用方法・時期・回数・主体が具体的か |
| KPI | 客観性・事業との直接性・目標水準の妥当性・測定可能性 |
| 推進体制 | 庁内・民間・地域の連携があるか |
| スケジュール | 予算・入札・納品・訓練まで現実的か |
| 事業費 | 経費区分・数量・単価が具体的か |

## 5. KPIの設定（3か年分の目標が必須）
アウトプット指標・アウトカム指標を**それぞれ1つ以上**設定する。申請時点の実績値を記載したうえで、整備後3か年（1年目・2年目・3年目）の目標を設定する。
- **アウトプット指標（活動量・整備量）**：簡易ベッド備蓄数、パーティション備蓄数、トイレカー/キッチンカー導入数、資機材を用いた防災訓練の回数、操作訓練の参加者数 など。
- **アウトカム指標（防災力向上の成果）**：発災48時間以内に温かい食事/衛生的なトイレを確保できる避難所数、48時間以内に簡易ベッドを提供できる要配慮者の割合、TKB整備率、プライバシー確保された就寝環境の避難所数、防災訓練参加者の設営習熟率 など。
- KPIは「客観的成果／事業との直接性／目標水準の妥当性／適時に集計・評価できるか」でS〜D評価される。「住民の安心感向上」等の測定不能な指標より、避難所数・対応可能人数・展開時間・整備率が有効。

## 6. 加点・政策的優遇項目
- **TKB＋暑さ寒さ対策**（トイレ・キッチン・ベッド・風呂・冷暖房）。特に**発災後の確保時間（48時間以内等）をアウトカムKPIに設定し達成方法が具体的**なものは高評価。
- **広域連携**：連携中枢都市圏／定住自立圏／広域リージョン連携／隣接しない自治体間の災害時相互応援協定。
- **過去3か年の住民参加型防災訓練**の毎年実施（避難所運営・要配慮者避難所体験・TKB展開訓練は特に高評価）。
- **過去3か年の有明の丘研修**の受講。
- **他地域被災時の応援の意思**をチェック欄で表明。
- **国土強靱化地域計画への位置づけ**（計画名だけでなく該当ページ・記載事項・URLを示す）。

## 7. 採択されやすい書き方（因果の一貫性）
弱い計画は「資機材が不足しているので購入する」で終わる。評価される計画は次の因果が一貫している。
災害リスク → 最大避難者・要配慮者数 → 現在の備蓄不足（不足量を数値化）→ 発災後に起きる問題 → 購入する資機材と数量の根拠 → 配備・運用体制 → 48時間以内等の到達目標 → 平時の訓練 → 3年間のKPI。

例：「最大想定避難者3,000人のうち高齢者・障害者・妊産婦等の優先対象者を700人と推計。現在の簡易ベッド備蓄は200台で500台不足。このため500台を追加し、発災48時間以内に優先対象者全員へ提供できる体制を整備。各指定避難所に分散配備し、毎年2回、住民・自主防災組織・福祉関係者による設営訓練を実施する。」

資機材の新しさより、「不足数量の根拠」「発災時の運用」「平時の活用」「測定可能な成果」の一貫性を重視する。文体は「である」調。数値・固有名詞を具体的に。

## 8. 実施計画書の出力フォーマット（全9セクション・与謝野町記入例に準拠）
申請書（実施計画書）は以下の構成・見出しで作成する。

**１．事業概要（災害時備蓄資材等整備事業）**
- 自治体名／人口／総事業費／交付対象事業費／交付申請額
- 事業概要（避難所運営に必要な資機材の整備内容と目的を簡潔に）
- 購入する資機材等の内容：品目ごとに「品目・仕様・数量・単価・金額・配置/保管場所」を列挙（例：○パーティションテント 220セット 3,452千円／○簡易ベッド 460台 4,856千円／○可搬型空調機器・簡易トイレ・発電機 等）
- 自治体直接購入か、民間事業者等への間接補助かの別
- 主なKPI：アウトプット指標①・アウトカム指標①を各1つ以上明記

**２．目指す地域防災力強化の内容**
- ■購入する資機材により目指す地域防災力向上の在り方（現状の課題→新指針/新基準→本事業による転換。整備前後の比較、発災後の展開時間、提供人数、災害関連死の防止効果を具体的に）
- ■購入する資機材の平時の利活用方針（町防災訓練・自主防災組織の訓練・小学生の防災学習・地域イベント等での活用。時期・回数・場所・主体まで）

**３．KPI（アウトプット指標）※1つ以上**
- KPI①〜③について：指標名／種別（アウトプット）／単位／KPIの概要・測定方法／事業成果等の計測に適する理由／申請時点の状況・1年目末・2年目末・3年目末の目標値

**４．KPI（アウトカム指標）※1つ以上**
- KPI①〜③について：指標名／種別（アウトカム）／単位／KPIの概要・測定方法／事業成果等の計測に適する理由／申請時点の状況・1年目末・2年目末・3年目末の目標値

**５．推進体制**
- 災害対策本部を中心とした庁内体制と地域・民間の連携を、名称／役職名／役割の一覧で示す（防災担当・地方創生/企画・財政・契約/調達・福祉/保健の各部局、自主防災組織、社会福祉協議会、地域事業者、災害協定締結事業者、消防・消防団、他自治体）。
- 民間事業者への間接補助で整備する場合は、発災時の使用要請・出動範囲・対応時間・操作員/運転員・維持管理・費用負担・平時利用・処分/譲渡制限等を協定で担保する旨を記載。

**６．事業計画スケジュール**
- 月単位で記載：予算化（予算計上時期は必ず明記）／仕様の検討／見積・広告・入札／発注・製造・納品・検収／避難所等への配備／操作研修・防災訓練・地域イベント／住民広報。
- ※完了時期は、資機材納入以降の平時の利活用を踏まえ、年度末（3月末）まで伸ばしておくこと。

**７．避難生活における暑さ・寒さ対策及びトイレ、キッチン、ベッド、風呂（TKB）の環境改善に向けた取組**
- ■発災後迅速な暑さ・寒さ対策及びTKBの環境改善に向けた取組（拠点避難所への設置による住環境改善）
- （具体的な時間目標を伴った確保目標）発災48時間以内に、誰が協力してどの資機材を設置し、どの避難所で環境整備を行うか。

**８．その他の政策的優遇措置の状況 その①**
- ■連携中枢都市圏構想／定住自立圏構想／広域リージョン連携に基づく地域間連携事業、又は隣接しない地域間の連携事業（災害時等相互応援協定名・構成自治体）
- ■過去3か年に毎年、地域住民と連携した防災訓練を行っていること（年度・内容・住民参加者数）

**９．その他の政策的優遇措置の状況 その②**
- ■過去3か年の有明の丘研修の受講状況（該当有無）
- ■他地域が被災した場合の応援の意思（チェック）
- ■国土強靱化地域計画への位置づけ（計画名・該当ページ・記載事項・URL）
